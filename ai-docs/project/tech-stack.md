# Tech Stack

## Core Technologies

- **Framework**: Next.js 15.2.4 with App Router
- **Runtime**: React 19
- **Language**: TypeScript 5
- **Styling**: Tailwind CSS 3.4.17
- **Package Manager**: pnpm

## UI Components

- **Component Library**: Shadcn UI (based on Radix UI primitives)
- **Accessibility**: Built-in via Radix UI primitives
- **Icons**: Lucide React
- **Animation**: Framer Motion, tailwindcss-animate

## Form Management

- **Form Handling**: react-hook-form 7.54.1
- **Validation**: Zod 3.24.1
- **Form-Zod Integration**: @hookform/resolvers 3.9.1

## UI Component Primitives (Radix UI)

All major Radix UI components are available:
- Accordion, Alert Dialog, Aspect Ratio, Avatar
- Checkbox, Collapsible, Context Menu
- Dialog, Dropdown Menu, Hover Card
- Label, Menubar, Navigation Menu
- Popover, Progress, Radio Group
- Scroll Area, Select, Separator
- Slider, Slot, Switch
- Tabs, Toast, Toggle, Toggle Group, Tooltip

## Additional UI Libraries

- **Command Palette**: cmdk 1.0.4
- **Date Handling**: date-fns 4.1.0, react-day-picker 8.10.1
- **Carousels**: embla-carousel-react 8.5.1
- **Toasts/Notifications**: sonner 1.7.1
- **Resizable Panels**: react-resizable-panels 2.1.7
- **Charts**: recharts 2.15.0
- **Drawer**: vaul 0.9.6
- **Typewriter Effect**: react-simple-typewriter

## Utility Libraries

- **CSS Utilities**: class-variance-authority 0.7.1, clsx 2.1.1, tailwind-merge 2.5.5
- **Theming**: next-themes 0.4.4
- **OTP Input**: input-otp 1.4.1
